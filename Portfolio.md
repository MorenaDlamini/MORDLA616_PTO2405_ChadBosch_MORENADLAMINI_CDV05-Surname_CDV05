## Overview
This is a simple developer portfolio website for showcasing projects, skills, and contact information. It demonstrates web development skills, user experience design, and project presentation.

## Entry Point
- The main entry point of the website is `index.html`.
- Styles are applied using `styles.css`.
- Interactive features like smooth scrolling and fade-in effects are handled by `script.js`.

## Features
- **About Section**: Brief introduction about the developer.
- **Projects Showcase**: Displays key projects with descriptions and images.
- **Contact Information**: Provides a way for visitors to reach out.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Usage
To view the portfolio:
1. Open `index.html` in a web browser.
2. Navigate through sections using the smooth scrolling feature.
3. Hover over buttons and links to experience interactive elements.

## Folder Structure
```
/portfolio
│── index.html          # Main entry point
│── styles.css          # Styling for the website
│── script.js           # JavaScript functionalities
│── images/             # Contains all images used in the project
│── README.md           # Project documentation
```
